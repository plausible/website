You can now break down the **Top Pages**, **Entry Pages** and **Exit Pages** reports by URL instead of just path.

This is useful if you track multiple subdomains in one dashboard. Pages with the same path on different subdomains are grouped together by default. For example, `yourdomain.com/best-page/` and `docs.yourdomain.com/best-page/` both appear as `/best-page/`. Breaking down by URL separates them so you can compare each page without filtering by hostname.

Open the **⋮** menu in the top right of the report and select **URL**. We’ll also save your choice for subsequent visits.

[Learn more about tracking subdomains](https://plausible.io/docs/subdomain-hostname-filter).
