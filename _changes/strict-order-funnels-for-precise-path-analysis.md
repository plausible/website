title: Strict order funnels for precise path analysis
date: 2026-04-14T07:18:43.983Z
---
You can now make your funnels strict.

When enabled, every step must happen in exact consecutive order with no other actions in between.

This is useful when you want to measure a tightly defined path and ensure any detour disqualifies the conversion.

Learn more in [our funnel analysis docs](https://plausible.io/docs/funnel-analysis).
